Description:
A simple implementation of the Finite Element Method, using tri-node elements, for the analysis of a welded steel pipe under plane-stress condition.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Instructions: 
To define pipe and weld geometry, copy the Nodes and Element data from AbaqusCAE '.inp' file into 'nodes.txt' and 'elements.txt' files respectively, and modify the required data in the premable of th emain '.py' file. 
Run the 'varjena_cev_FEM_working.py' file.
